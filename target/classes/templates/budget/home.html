Guia para controle de orçamentos

Home de Orçamentos

Ao selecionar a aba "Orçamentos" o programa irá redirecionar o usuário para a página inicial de orçamentos, no qual contém os orçamentos ja cadastrados e salvos no sistema.

O primeiro botão da tela, que possui a função de direcionar o usuário para a tela de cadastro

No final de cada linha orçamento na relação em forma de tabela, temos a coluna "Gerenciar", que possui os seguintes botões.

O primeiro botão tem por função direcionar o usuário para a tela de visualização daquele orçamento específico.

O segundo botão direciona o usuário para a tela de atualização, no qual os dados já cadastrados podem ser alterados.

O terceiro botão exclui o orçamento daquela linha em específico.

Cadastro de Orçamentos

Ao entrar na tela de cadastro de orçamentos o usuário poderá preencher os campos referentes às informações do orçamento.

A área de seleção é onde o usuário irá selecionar um produto cadastrado no sistema para incluir no orçamento.

O campo de texto ao lado é para o usuário colocar a quantidade daquele determinado produto escolhido na área de seleção a esquerda.

Para alterar o valor da quantidade de um produto já cadastrado, basta selecionar o produto na área de seleção e colocar a quantidade que deseja somar ou subtrair na quantidade.

- Um número positivo para aumentar a quantidade;

- Um número negativo para diminuir a quantidade.

O botão ao lado direito é para adicionar o determinado produto e a sua quantidade desejada e, caso o produto já esteja inserido no orçamento, então somente a sua quantidade será alterada.

O botão em cada linha de um produto inserido no orçamento tem como finalidade excluir aquele deterinado produto.
