Guia para controle de produtos

Segue um guia para orientar o usuário com relação aos botões presentes nas telas do STM para o controle de produtos, cada botão será explicado.

Entretanto vale salientar que se o botão explicado aparecer mais de uma vez em telas diferentes, o mesmo possui a mesma função.

Home de produtos

Ao selecionar a aba "Produtos" o programa irá direcionar o usuário para a página inicial de produtos que contém uma relação com todos os produtos cadastrados no sistema e também com alguns botões.

O primeiro botão da tela, que possui função de direcionar o usuário para a tela de cadastro

No final da linha de cada produto na relação em forma de tabela, temos a coluna denominada "Gerenciar", que possui os seguintes botões.

Segue o primeiro da coluna gerenciar que possui a função de direcionar o usuário para a tela de visualização do produto em questão.

O segundo botão possui a função de direcionar o usuário para a tela de atualuzação (ou alteração de dados) do produto em questão.

O terceiro e último botão possui a função de deletar o produto em questão.

Cadastro de produtos

Ao entrar na tela de cadastro o usuário terá como preencher campos referentes às informações do produto como segue abaixo.

Ao inserir as informações o usuário verá dois botões no final do formulário.

O primeiro possui a função de confirmar o cadastro

Já o segundo possui a finalidade para limpar todo o cadastro, para recomeçar o mesmo do zero.

Visualização de produto

Ao selecionar o botão para visualizar o produto as informações aparecerão para o usuário, porém sem a possibilidade de alterá-las.

Após visualizar o produto e se desejar retornar para a página inicial o botão abaixo realiza essa tarefa.

O botão voltar está presente em todas as telas do controle de produtos com exceção da inicial.

Atualização de produto

Após selecionar o botão de atualização de produto as informações do produto selecionado aparecerão e também poderão ser alteradas.

Para confirmar as alterações o botão abaixo possui a finalidade de realizar essa tarefa.
